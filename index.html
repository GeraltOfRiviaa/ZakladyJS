2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. JavaScript is case sensitive - "This" is not the same as "this"
  2. Commands must be terminated with a semicolon if there are multiple commands on one line (termination semicolon is also appropriate for single commands)
  3. The block of commands is enclosed in parentheses - {...}
  4. Comments in JS: // one-line comment /* multi-line comment */
  5. JavaScript doesn't use normal variables like int, float, etc. but special variables like let and const. The difference is that JavaScript only makes a difference between the variable, that will change its value (let) and that will not change its value (const)

3: Jak je možné propojit JavaScript s webovou stránkou?

Doplňte další možnosti ve formě kódu.

Opatřete kód poznámkami!

1. Vložit kód do tagu script
<script>console.log('Hello');</script>
2. Připojit externí soubor
<script type= 'text/javascript' src='soubor.js!><>/script;>
3. Vložit kód jako reakci na události
<button onclick="alert('Ahoj'); ">Klikni</button>

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // nullová hodnota
let x = 1 // celé číslo (typ number)
let x = 1.5 // reálné číslo (typ number)
let x = 165e-5 // zápis desetinného čísla pomocí exponentu
let x = 'text' // typ string
let prestupny = false // typ boolean

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění), ++ (+1), -- (-1)

Přiřazovací

=, +=, -=, *=, /=, %= (stejné jako x = x % 5), **= (stejné jako x = x ** 5)

Logické

%% (a), || (nebo), ! (ne)

Porovnávací

==, === (zkontroluje, jestli je stejná hodnota a operátor), !=, !== (nerovná se hodnota a typ), >, <, >=, <=, ?

Bitové

& (AND), | (OR), ~ (NOT), ^ (XOR), << (levý shift), >> (pravý shift), >>> (unsigned right shift)

Typové

typeof (typ proměnné), instanceof (objekt je instancí typu objekt)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Uživatel zadal špatně heslo */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt(Zadej heslo);

confirm()

/* Popup okno se uživatele zeptá, zda chce na stránky seznamu */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování: Při použití popup oken nemůže uživatel na stránce dělat nic, dokud se popup okno nezavře
